# Hex to rgb
A simple web app for converting between hex and rgb.

## Usage
1. Clone repo
    ```bash
        git clone https://github.com/Matmonsen/hex_to_rgb.git
        cd hex_to_rgb
    ```
2. Install dependencies
    ```bash
        npm install
    ```
3. Run local dev server
    ```bash
        npm run dev
    ```
4. Open browser at http://localhost:1337/
